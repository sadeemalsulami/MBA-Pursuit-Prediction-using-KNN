
# MBA Pursuit Prediction using KNN
##  Project Overview
This project aims to analyze factors influencing the decision to pursue an MBA after completing a bachelor’s degree. Using data science techniques, the project identifies patterns and builds a predictive model to classify whether a student is likely to pursue an MBA and determine their preferred study mode (Online or On-Campus).
The project demonstrates the full data science workflow, including data cleaning, exploration, modeling, and evaluation.
---
##  Objectives
- Analyze student-related factors affecting MBA decisions
- Clean and preprocess raw data
- Explore relationships between features
- Build a classification model using K-Nearest Neighbors (KNN)
- Evaluate model performance using standard metrics
- Generate insights useful for educational institutions
---
##  Data Preprocessing
The dataset underwent several preprocessing steps:
- Handling Missing Values  
  Missing categorical values were replaced with "Unknown"
- Removing Duplicates  
  Duplicate rows were removed to ensure data quality
- Dropping Irrelevant Features  
  Unnecessary columns were removed to improve model performance
---
##  Exploratory Data Analysis (EDA)
Key steps included:
- Calculating summary statistics (mean, standard deviation)
- Identifying patterns between features
- Understanding relationships in cleaned data
---
##  Model Development
### Algorithm Used:
K-Nearest Neighbors (KNN)
KNN was selected because:
- Simple and effective for classification
- Works well with mixed data types
- Suitable for smaller datasets
### Data Splitting:
- 80% Training Data
- 20% Testing Data
---
##  Model Evaluation
The model was evaluated using:
- Confusion Matrix
- Classification performance analysis
The model achieved reasonable accuracy and demonstrated the importance of feature selection and preprocessing.
---
##  Results and Insights
- The KNN model successfully identified patterns in MBA decision behavior.
- Online MBA programs appear to attract a significant portion of students.
- Feature selection plays a critical role in model performance.
